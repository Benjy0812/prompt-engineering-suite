# Technology Stack - Prompt Engineering Suite

## 1. Core Format

- **Markdown (CommonMark/GFM)**: The primary language for all prompt templates
  and documentation.

## 2. Infrastructure & Automation

- **Shell Scripting (Bash)**: Used for maintenance tasks, automated prompt
  generation, and suite management.
- **Git**: Primary version control system for tracking prompt versions and
  contributions.

## 3. Formatting & Linting

- **Prettier**: The standard tool for ensuring consistent formatting across all
  Markdown and JSON files in the repository.

## 4. Target LLM Models (Primary Verification)

- **Google Gemini Family**: Optimized and tested for Gemini 1.5 Flash, 1.5
  Flash-Lite, and 1.5 Pro.
- **Anthropic Claude**: Support for Claude 3.5 Sonnet and other major models.

## 5. Ecosystem & Tooling

- **CLI Tools**: Designed for use with `gemini-cli`, `claude-code`, and similar
  LLM command-line interfaces.
- **IDEs**: Compatible with VS Code (via extensions or manual use) and other
  development environments.
- **Web Interfaces**: General compatibility with browser-based LLM chat
  interfaces.
